Usage
1Go to Profile → add API key → paste CV → Extract from CV
2Go to Evaluate → paste any JD → tap Evaluate Job
3Read your Score + GLS → decide whether to apply
4Save to Tracker → manage your full pipeline
💡 Decision guide

Score ≥ 4.0 + GLS ≤ 50 → Apply now  ·  Score ≥ 4.0 + GLS > 50 → Verify role is active first  ·  Score < 3.5 → Skip

1Go to Profile → add API key → paste CV with projects and CGPA
2Go to Evaluate → toggle to Student/Intern → paste internship JD
3Check your PPO score — the internship IS the job interview for most students
🚀 One tip

After your first evaluation, ask: "I keep getting low Project Match scores. What should I build next?" The system will suggest the specific project that closes the gap fastest.
